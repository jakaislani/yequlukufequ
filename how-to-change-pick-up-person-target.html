How do I change my target pick up drive up?

Can I change my Drive Up or Order Pickup after the order is placed? Once you receive notification that your order is ready to be picked up, you can change your pick up method from Order Pickup to Drive Up or Drive Up to Order Pickup using the Target app. Use the prompts in the app to select your desired pickup method.

How do you change a pickup person?

If you’ve already checked out but would like to either add or change the existing pickup person:
  • Sign in to your account.
  • Select the account button.
  • Next to the order select See Details.
  • Select Edit next to the Current Pickup Person.
  • Add your alternate pickup person information.
